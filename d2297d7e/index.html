03

The privacy picture, honestly

You chose public links with no password — the simplest path. Here's exactly what that protects and the one trade-off that comes with going free.

What's covered

protected
Showing up in GoogleNo — noindex + robots
Guessable by strangersNo — random slug
Shareable with anyoneYes — just send the link
CostFree, forever

The trade-off

free-tier reality
Repo visibilityPublic
Raw files findableYes, if someone finds the repo
Anyone gets blocked outNo — nothing is auth-walled
Fix availableYes — a password layer

The only soft spot: free GitHub Pages requires a public repo, so the raw HTML is technically visible at github.com/Mullermtg/pages to anyone who finds it. The random slugs and noindex keep pages out of search and off the radar, which covers normal sharing. If you ever want them genuinely unreadable, the documented next step is a StatiCrypt password — it AES-encrypts each page so the repo can stay public and free while the content is a locked blob without the key.

04

Using it, day to day

Three things you'll actually do. None of them involve a dashboard, a login, or a config file.

Make one

Just ask for a page like you always have. The publish step is wired into the skill, so it runs automatically. Say "local only" if you ever want to skip it.

Share one

The link is already on your clipboard when the page opens — mullermtg.github.io/pages/<slug>/. Paste it to a client or teammate. It just works in their browser.

Revisit past work

Open the dashboard at mullermtg.github.io/pages/. Everything you've made is listed newest-first with its date and a one-click link.

# and if you ever want to publish a file by hand:
$ ~/.claude/skills/html-creator/publish.sh ~/Downloads/my-page.html "My Page Title"

The takeaway

You now have a self-updating archive with shareable links, built and tested entirely from the terminal — no dashboards, no monthly cost. The page you're reading is the first real one through the pipeline; it's listed on your dashboard right now. If "public repo" ever starts to bug you, the password upgrade is a small, documented change away — just say the word.
